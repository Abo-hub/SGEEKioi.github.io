2. 市场分析

2.1 市场环境

中国短视频用户规模2019年约达到6.27亿人，短视频市场规模突破400亿元。短视频内容的丰富性和多变性可以提供更好的内容体验。随着近几年短视频行业的飞速增长，短视频行业发展已经步入成熟期，但行业发展前景依然可期。

(数据来源-中国产业信息网)

短视频与其他内容产品的区别：

  • 不涉及版权问题：相比于长视频电影或电视剧，通常需要平台购买版权或平台自制原创内容来吸引用户付费。
  • 内容丰富，浏览时间碎片化：短视频与图文内容相比，内容上更加丰富观感更好。并且视频包罗万象，既有搞笑型、才艺型、技术流等内容。
  • 社区化：视频内容由用户创作，

2.2 产品定位

抖音是一个帮助用户表达自我，记录美好生活的短视频平台。由于其社交互动的属性拥有更强的用户粘性和用户的吸引力，2020/1/5日，抖音日活跃用户已超4亿。

(数据来源-2019抖音数据报告)

3. 用户分析

3.1 抖音的用户属性

(数据来源-艾瑞数据)
(数据来源-易观千帆 )

根据艾瑞数据2020年1月的数据，抖音的用户男性略多于女性。使用人群年龄占比中，24岁及以下占比最高，达到了31.54%。30岁以下的用户占到了56.35%。该年龄段是短视频群体的主力军，是内容的主要生产者。随着抖音用户的不断增多，31-35岁用户占比也达到了25.92%。

3.2 抖音的用户分类

根据长时间使用，总结抖音用户群体可以分为五类：

  • 专业生产者：主要是专业自媒体团队或者MCN机构，这些用户是主要的内容生产者，并且和官方合作，以带动用户的活跃度。
  • 企业用户：主要是企业和商家以宣传销售为目的。作为短视频类明星产品，自然越来越多商家入驻抖音进行营销推广。
  • 官方账号：各大新闻媒体的官方账号，通过这些官抖发布的内容，我们可以了解到更加准确的新闻和实时热点。
  • 普通创作者：会受到KOL的影响去创作视频拍同款，或者分享自己的生活。
  • 娱乐消遣：以观看短视频为主的吃瓜群众。很少创作视频，更多是关注有趣或者热点的事情。

3.3 用户需求分析

根据以上用户分类，可以总结出抖音用户的不同需求

  1. 获取收益
    以网红类用户为代表，通过平台获得知名度和粉丝，可通过直播、带货、商务合作获取收益。这类用户通常有自己的团队，负责剧本、拍摄、后期剪辑，可产出高质量的作品，有一定话题热度，可拍出爆款视频。
  2. 营销推广
    抖音作为一个dau超过4亿的流量新星，自然被越来越多商家作为产品推广的渠道，团队自己创作短视频为店铺引流(代表：牛肉哥)、或在平台投放广告(app推广)、找网红进行合作推广等。
  3. 娱乐消遣
    抖音用户的一个核心的需求，打发碎片化时间，观看推荐的内容或热点，是内容的消费者
  4. 分享生活
  5. 信息获取

4. 功能分析

4.1 观看视频

推荐页是浏览视频的一个核心入口，更是整个抖音的核心。抖音的视频占据一整个屏幕，营造了更加沉浸式地体验，上下滑动切换视频，方便的手势操作(双击点赞，长按)，再加上字节强大的推荐系统，可能只是不断地下滑观看内容，时间已经过去了1个小时(如果你刷的时间过长，抖音官方会有提示你休息地视频，别问我怎么知道的)。

打开抖音app，经过开屏广告，默认进入推荐页，视频占据一整个屏幕，用户只需上下滑动即可切换视频，营造了更加沉浸式的体验。

运营分析

总结

优点

  • 推出视频制作工具“剪映”
  • 推出“星图”，星图平台是品牌主、MCN公司和明星/达人进行内容交易的服务平台，有官方平台保障。
  • 所有内容通过首页推荐分发，交互简单，用户直接消费内容价值

缺点

  • 热榜过于单一，并且同质化严重。
