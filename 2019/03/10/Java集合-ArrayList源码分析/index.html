ArrayList简介

ArrayList的底层是数组队列，相当于动态数组。在添加大量元素前，应用程序可以使用==ensureCapacity== 操作来增加 ArrayList 实例的容量。这可以减少递增式再分配的数量。

它继承于AbstractList，实现了List、RandomAccess、Cloneable、java.io.Serializeble这些接口。

  • ArrayList继承于AbstractList。实现了List。实现了所有可选列表操作，添加、删除、修改、遍历等功能。
  • ArrayList实现了RandomAccess。RandomAccess是一个标志接口，用来表明其支持快速（通常是固定时间）随机访问。在ArrayList中，我们可以通过元素的序号快速获取元素对象，这就是快速随机访问。
  • ArrayList实现了Cloneable接口，覆盖了函数clone(),能被克隆
  • ArrayList实现了java.io.Serializable接口，意味着ArrayList支持序列化

区别：
